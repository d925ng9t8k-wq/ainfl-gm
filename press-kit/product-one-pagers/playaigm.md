# PlayAiGM — Product Brief

**Status:** Live | **URL:** https://ainflgm.com | **Category:** Consumer AI / Sports Entertainment

---

## What It Is

PlayAiGM is an AI NFL game management simulator. Users take the role of an NFL General Manager — drafting players, managing the salary cap, making trades, and simulating seasons — powered by real NFL data updated twice daily.

## Target User

NFL fans aged 18–45. Fantasy football players. Sports bettors researching team fundamentals. Anyone who has ever yelled at their team's front office.

## Core Features

- AI-powered trade analyzer with real salary cap data
- Full NFL draft simulator with player grades
- Season simulator — project your team's record
- War Room — 32-team needs analysis
- Matchup predictor, cap space explorer, injury impact tool
- 50+ NFL tools accessible from a single dashboard
- PWA installable — works offline, feels like an app

## Technology

- Built with Vite + vanilla JS
- Deployed on GitHub Pages with CI/CD via GitHub Actions
- Data: OTC (Over the Cap), PFF grades, public NFL APIs
- Data refresh: 2x daily automated cron jobs
- Google Analytics: G-PLW4H1NNF6

## Business Model

- Google AdSense (pending account approval — applied April 2026)
- Affiliate revenue: BetMGM, FanDuel banners (in plan for NFL Draft window, April 23-25)
- Free product — no subscription, no paywall
