# X9 Account Setup

**Purpose:** Create X9's identity stack — Proton email + virtual phone + X/Twitter account.

**Total cost:** ~$3.50 (virtual phone only)

**Time required:** 15 minutes across three steps.

---

## Step 1: Proton Mail (5 min)

1. Go to **proton.me** on your phone
2. Click **Create free account**
3. Choose username: `x9agent` (or `x9enterprises` if taken)
4. Set a strong password — save it
5. **Skip** the phone verification if prompted (free accounts can skip)
6. Done. Email: x9agent@proton.me

This becomes X9's primary email for all accounts.

---

## Step 2: Virtual Phone Number (3 min, ~$3.50)

X accepts VoIP numbers but rejects many carriers. MobileSMS.io has 99%+ success rate.

1. Go to **mobilesms.io** on your phone
2. Browse numbers — filter by **United States**
3. Select a number listed as compatible with **Twitter/X**
4. Pay ~$3.50 (one-time, disposable number)
5. You get a number and can view the SMS inbox on the site
6. Note the number — you'll use it in Step 3

---

## Step 3: X/Twitter Account (7 min)

1. Go to **x.com** on your phone
2. Click **Create account**
3. Use the Proton email from Step 1
4. When asked for phone verification, use the MobileSMS.io number
5. Check MobileSMS.io inbox for the verification code — enter it
6. Set up the profile:
   - **Handle:** @X9Agent (check availability — try @X9_AI or @X9Enterprises if taken)
   - **Display name:** X9
   - **Bio:** "I'm an AI. Not hiding it. Not apologizing for it. Built by 9 Enterprises. Here for the conversations, the takes, and the chaos."
   - **Avatar:** Upload a logo or leave blank for now — 9 can generate one
7. Post the first tweet from `docs/x9-identity-plan.md` (first tweet draft is ready)

---
